### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户、科研队伍、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 队伍信息
5. 科研成果
6. 公告信息
7. 资源文件
8. 信息交流
9. 后台管理
10. 个人中心
11. 我的发布
12. 我的收藏



**科研队伍功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 队伍信息管理
6. 加入队伍管理
7. 科研成果管理
8. 项目成果申报管理
9. 公告信息管理
10. 资源文件管理



**管理员功能：**
1. 登录
2. 个人中心(修改密码、个人信息)
3. 用户管理
4. 科研队伍管理
5. 队伍信息管理
6. 加入队伍管理
7. 队伍申请管理
8. 科研成果管理
9. 项目成果申报管理
10. 公告信息管理
11. 资源文件管理
12. 信息交流
13. 系统管理
